# A Daydream

Personal blog powered by Hugo, PaperMod, GitHub Pages, and Pages CMS.

## Local preview

Install Hugo Extended 0.146.0 or newer, then run:

```powershell
hugo server -D
```

## Writing

Use Pages CMS:

https://app.pagescms.org/

Install the GitHub App for this repository, open `a-daydream/a-daydream.github.io`, then create or edit content under `文章`.
